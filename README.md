# Czyszczenie i analiza zamówień e-commerce

Autor: Amelia Danił  
Numer albumu: 159803

## Opis projektu

Projekt przedstawia proces czyszczenia, transformacji oraz analizy danych dotyczących zamówień e-commerce przy użyciu biblioteki pandas.

## Zakres projektu

Projekt obejmuje:

- generowanie brudnych danych,
- eksplorację i identyfikację problemów jakości danych,
- czyszczenie danych,
- transformacje danych,
- analizę SQL-style z użyciem groupby,
- wizualizację danych,
- zapis oczyszczonego zbioru do pliku CSV.

## Wykorzystane biblioteki

- pandas
- numpy
- matplotlib
- re
